Smash\+The\+Mole is a Whac-\/\+A-\/\+Mole style game built in C++ using SFML 2.\+x. It features\+:
\begin{DoxyItemize}
\item A start menu with a custom background.
\item Gameplay with a separate background image.
\item Targets (moles and bombs) that appear on a grid.
\item A custom cursor (whacker) replacing the system mouse cursor.
\item Dynamic scoring and a countdown timer.
